# MAPP E2E testsuite

E2E test service for Smartpixel and SPA plugins.

## Commands

Start the service: `make start_server`

Stop the service: `make stop_server`

Start the server with interactive CLI mode: `make debug_server_start` You can quit by pressing CRTL + C twice, however the DB will still be running.

Start / stop the database: `make db_start`  `make db_stop`
