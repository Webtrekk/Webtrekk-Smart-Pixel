A Angular implementation of the Webtrekk Smart Pixel.

## Installation

Using npm:

```sh
$ npm install --save @webtrekk-smart-pixel/angular
```

## Integration

```js
import * as wtSmart from '@webtrekk-smart-pixel/angular';
```
